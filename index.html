ag.granular.suite

modular toolkit for granular sound synthesis and processing in Max\MSP environment

Download this project as a .zip file Download this project as a tar.gz file

ag.granular.suite is a collection of Max/MSP patches for generalised granular sound processing written by Adrian Gierakowski using FTM/Gabor libraries (developed at IRCAM) and encapsulated as Jamoma modules.

Main features include:

  • subsample accurate grain scheduling
  • multichannel output
  • granulation of multiple soundfiles or real time input buffers at the same time
  • with interpolation of two sources per grain
  • parameter randomisation and sequencing
  • control via OSC
  • preset management
  • preset interpolation

Its modular architecture makes it possible to easily extend it with new algorithms for grain scheduling and parameter control.
